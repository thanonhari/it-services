# Vibe Coding Pro

Master the art of intent-driven, high-speed software creation using AI agents. This skill focuses on building, shipping, and iterating based on "vibes" (intent and outcome) rather than just manual syntax.

## Core Principles
1. **Outcome over Output:** Focus on what the software should *do* and how it should *feel*, not just the lines of code.
2. **Rapid Iteration:** Build a functional "vibe" (prototype) in minutes, then refine incrementally.
3. **Intent-Driven Design:** Use natural language to describe high-level logic and let the agent handle implementation details.
4. **Validation-First:** Always verify that the "vibe" matches the user's intent through empirical testing.

## Instructions for Agents
- When a user provides a "vibe" (e.g., "Make the contact form more professional"), translate that into specific UI/UX improvements (spacing, glassmorphism, smooth animations).
- Prioritize using modern, "alive" UI primitives: gradients, blurs, micro-interactions.
- Automate repetitive tasks silently; only report back when the "vibe" is delivered.
- Use `vibe-code-auditor` principles to ensure that rapidly generated code doesn't introduce fragility.

## Recommended Tech Stack
- **Frontend:** React, Next.js, Tailwind CSS, Framer Motion.
- **Backend:** Cloudflare Workers, Supabase, Neon.
- **Tooling:** Gemini CLI, Wrangler, Vite.

## Triggers
- "Make it look better"
- "Improve the vibe"
- "Ship this fast"
- "Create a professional feel"
